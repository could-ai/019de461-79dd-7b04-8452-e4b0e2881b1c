# AI Создатель Приложений

AI Создатель Приложений — это мощная среда разработки (IDE) на базе искусственного интеллекта без цензуры, которая позволяет создавать кроссплатформенные приложения и игры с помощью естественного языка.

## Обзор продукта

Приложение предоставляет интерфейс для общения с продвинутым ИИ, который берет на себя всю работу по написанию кода. У ИИ нет ограничений: он может сгенерировать приложение любой сложности и сразу сообщить, сколько времени займет его разработка. 

Пользователи могут не только генерировать приложения, но и:
- Просматривать и редактировать исходный код, написанный ИИ.
- Тестировать сгенерированное приложение во встроенном превью.
- Настраивать параметры сборки: имя файла (APK), иконку на рабочем столе и уровень графики (от низкого до ультра) для создаваемых игр.

## Основные функции
- **Чат с ИИ-разработчиком:** Прямое общение с ИИ для описания логики, дизайна и механик вашего будущего приложения.
- **Редактор кода:** Полноценный текстовый редактор для ручной доработки сгенерированных исходников.
- **Интерактивное превью:** Виртуальный экран смартфона для мгновенного тестирования созданного приложения.
- **Настройки сборки (Settings):** Выбор иконки приложения, названия APK-файла и предустановки качества графики для игровых проектов.

## Пользовательские сценарии
1. Пользователь открывает приложение и переходит на вкладку **Чат**.
2. Описывает свою идею: "Создай игру-платформер с главным героем-котом".
3. ИИ принимает задачу, сообщает точное время выполнения (например, 45 секунд) и генерирует проект.
4. Пользователь переходит на вкладку **Код**, чтобы просмотреть написанное, и может внести свои правки.
5. Открыв вкладку **Превью**, пользователь тестирует игру в реальном времени.
6. На вкладке **Настройки** пользователь задает имя файла (например, `CatPlatformer`), выбирает иконку и устанавливает уровень графики (например, "Ультра").
7. Пользователь нажимает "Собрать APK" и получает готовое приложение.

## Технологический стек
- Фреймворк: [Flutter](https://flutter.dev) (Dart)
- Платформы: Web, iOS, Android, macOS, Windows, Linux

## Инструкция по запуску
1. Убедитесь, что у вас установлен Flutter SDK.
2. Клонируйте репозиторий и перейдите в папку проекта.
3. Выполните команду `flutter pub get` для установки зависимостей.
4. Запустите проект командой `flutter run`.
